# 작업 완료 시 체크리스트

## 코드 작성 후 필수 실행
1. **타입 체크**: `npm run type-check`
2. **린팅**: `npm run lint` (자동 수정: `npm run lint:fix`)
3. **테스트**: `npm test` (새로운 기능이면 테스트 코드 작성)
4. **빌드 확인**: `npm run build`

## Git 커밋 전
- Husky pre-commit 훅이 자동으로 lint-staged 실행
- 모든 TypeScript 에러 해결
- 테스트 통과 확인

## 코드 리뷰 포인트
- PWA 기능 영향 확인 (오프라인, 백그라운드 동기화)
- 다국어 지원 고려 (한국어/영어)
- 모바일 반응형 확인
- 접근성 고려
- 성능 최적화

## 배포 전 확인사항
- 프로덕션 빌드 성공
- PWA 매니페스트 정상 작동
- 서비스 워커 등록 확인
- 다국어 번역 완료
- 메타데이터 설정 확인
